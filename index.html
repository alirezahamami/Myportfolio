Halifax Bus Map

The primary goal of the Bus project was to improve the accessibility and efficiency of the Halifax metro bus system through the creation of an interactive online platform. Utilizing the Leaflet library in JavaScript, the project aimed to offer users a real-time visualization of bus routes and movements on a dynamic map. Furthermore, this project had the potential to function as a tool for monitoring and managing traffic flow within the municipality of Halifax by tracking some or all of the bus routes.

3-In-A-Row Game

Developed during my time at NSCC, this dynamic 'Three in a Row' puzzle showcases my proficiency in front-end web development. Utilizing unobtrusive JavaScript, the puzzle's grid is generated based on JSON data fetched from a remote location. Players can interact with the puzzle by left-clicking to cycle through colors, and a 'Check Puzzle' button provides real-time feedback on the puzzle's completion status. An innovative feature enhances the experience, ensuring a seamless and engaging gameplay, while adhering to responsive design principles for various puzzle sizes.

Chinook Music Library

I developed a Node.js server to handle data from the Chinook Music Library, providing a robust backend solution for managing and querying music-related information. Leveraging Express.js for routing and interaction with the SQLite database, the server offers a RESTful API with endpoints for accessing artists, albums, tracks, playlists. Key features include implementation of various SQL queries to fetch and manipulate data, robust error handling, and input validation to ensure data integrity. This project showcases my proficiency in backend development with Node.js, database management, and REST API design, demonstrating my ability to deliver efficient and scalable solutions for handling complex data structures.
